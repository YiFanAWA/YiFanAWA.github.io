## 项目

### TCM-Eval - 中医大语言模型评测基准

本仓库包含了 [TCM-Eval](https://github.com/Zescy1/TCM-Eval) 项目的最新源代码（作为 Git 子模块）。

TCM-Eval 是一个用于评估大语言模型在传统中医领域性能的评测基准框架。它支持多种开源模型和 API 模型的评测。

#### 查看源代码

- 在线浏览：[TCM-Eval 目录](./TCM-Eval/)
- GitHub 仓库：[Zescy1/TCM-Eval](https://github.com/Zescy1/TCM-Eval)

#### 更新子模块

如果需要更新到最新版本的 TCM-Eval：

```bash
git submodule update --remote TCM-Eval
```
